# Security Policy

## Supported Versions

Currently, there are no released version and master branch is all that is supported.
In the future, the table below will be used to indicate supported versions for security disclosures.

| Version | Supported          |
| ------- | ------------------ |
| master  | :white_check_mark: |

## Reporting a Vulnerability

This document is still a work in progress and is not expected to provide comprehensive direction for disclosures at this time.
For any questions or disclosures, please email security@nullserve.com and work with one of our engineers directly.
